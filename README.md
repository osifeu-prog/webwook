# Telegram Git Bot - בוט לימוד מתקדם

בוט טלגרם לשימור הודעות, ניהול תלמידים, ומערכת מטבעות מתקדמת.

## ✨ תכונות חדשות

### 💰 מערכת תשלומים
- **תשלום 444 ש"ח** עבור גישה
- **אישור תשלום** עם צילום מסך
- **קישור אוטומטי** לקבוצה לאחר אישור

### 🪙 מערכת מטבעות
- **כריית מטבעות** על ידי מנהלים
- **העברת מטבעות** כתגמול למטלות
- **מעקב עסקאות** מלא
- **ארנק אישי** לכל משתמש

### 📁 ניהול תלמידים
- **תיקיות אישיות** לכל תלמיד
- **אישור גישה** עם תשלום
- **היסטוריית עבודה** מלאה
