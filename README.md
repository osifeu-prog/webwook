# 🤖 WebWook Bot - Telegram Reward System

בוט Telegram מתקדם למערכת תגמולים מבוססת blockchain עם משימות, הפניות וחלוקת טוקנים אוטומטית.

## ✨ תכונות

### 🎯 מערכת משימות
- 10+ משימות מוכנות עם תגמולים דינמיים
- הגשת הוכחות ואישור אוטומטי/מנהל
- מעקב התקדמות אישי

### 💰 חלוקת טוקנים אוטומטית
- אינטגרציה עם Binance Smart Chain (BSC)
- חלוקת טוקנים אוטומטית למשימות שאושרו
- ניהול ארנקים ומעקב עסקאות

### 👥 מערכת הפניות
- קישורי הזמנה אישיים
- תגמולים דו-שלביים (למזמין ולמוזמן)
- לוח מובילים והשוואות

### 🛡️ אבטחה ומנהל
- פאנל ניהול למנהלים
- ולידציית קלט ואבטחה
- logging מפורט וניטור
